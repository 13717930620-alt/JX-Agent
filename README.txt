🚀 快速开始（快速体验）：
  1. 确保已安装 Node.js (https://nodejs.org/) v18+
  2. 双击「启动.bat」选择模式直接运行
  3. 或者双击「安装程序.bat」安装到指定目录

🚀 快速开始（完整安装）：
  1. 确保已安装 Node.js (https://nodejs.org/) v18+
  2. 双击运行「安装程序.bat」
  3. 按提示操作（输入 API Key，选择模型）
  4. 启动后用 CLI 直接对话，或打开浏览器访问 http://localhost:3000

🌐 访问方式：
  • 网页版: http://localhost:3000
  • API:    POST http://localhost:3000/api/chat
